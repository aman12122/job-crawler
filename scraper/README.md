# Job Crawler - Scraper

Python web scraper for tracking job postings from company career pages.

## Setup

```bash
# Create virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install -e ".[dev]"
```

## Usage

```bash
# Run scraper (coming soon)
python -m src.main
```

## Development

```bash
# Run tests
pytest

# Lint code
ruff check src/

# Type check
mypy src/
```
